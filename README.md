# Fastlane CI/CD Skill for Claude Code

A Claude Code skill that helps set up Fastlane CI/CD pipelines with match code signing for iOS apps, including GitHub Actions workflows and TestFlight deployment.

## Installation

```bash
/install davetza/fastlane-cicd-skill
```

## Usage

Use `/fastlane-cicd` when:
- Setting up CI/CD for a new iOS project
- Migrating from manual builds to automated deployment
- Configuring TestFlight or App Store deployment
- Setting up code signing with match
- Creating GitHub Actions workflows for iOS

## What This Skill Provides

- Complete Fastlane configuration templates (Gemfile, Appfile, Matchfile, Fastfile)
- GitHub Actions workflow for automated builds
- Match setup for code signing certificate management
- TestFlight deployment configuration
- Troubleshooting guide for common issues
